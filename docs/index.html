Hierarchical Population Clustering

Interactive 3D visualizations of global population distribution using gravitational potential-based clustering.

This project models population as a gravitational field, where each person contributes to a "potential" that attracts others. The resulting 3D surfaces show population concentration as terrain - cities become mountain peaks, deserts become valleys.

About the Project

This visualization is part of a larger research project on hierarchical population clustering. The same algorithm that creates these 3D surfaces also generates spider-web network maps showing how populations merge hierarchically from neighborhoods to continents.

The algorithm discovers natural regional boundaries without prior geographic knowledge - it finds that Central Park divides Manhattan, that the Sahara separates Africa, and that the Grand Trunk Road corridor forms a coherent region spanning Pakistan, India, and Bangladesh.

Source code and documentation: GitHub Repository

Data source: Gridded Population of the World (GPW v4), 15 arc-minute resolution (~1 million grid cells)
